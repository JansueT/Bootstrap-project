## Learning objectives

- We understand the concept of CSS frameworks.
- We use documentation for CSS frameworks.

## Description

- In this exercise, we experience the power of using a CSS framework like Bootstrap to create a new website quickly from scratch. We are going to create a prototype of the portfolio website from scratch in a few hours.
